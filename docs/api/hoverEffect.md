# Hover Effects

This is an additional style option allowing to add hover effect when **user looks at an element**. It's applied to all Touchable and Pressable components by default.

```tsx
<TouchableOpacity style={{ cursor: "pointer" }}>
  <Text>Click me</Text>
</TouchableOpacity>
```
